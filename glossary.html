Glossary

Antivirus Software

Antivirus software is a program that you install on your computer, which will frequently scan to find any security threats in the files that are already on your computer as well as the ones, you download from the internet.

APK (Android Package Kit)

APK is the file format used on android to install applications.

Bandwidth

The bandwith is the maximum amount of data that can be sent on an internet connection at the same time.

Cache

Cache is a small amount of memory that is used to store recently open, or frequently accessed files, so that your computer doesn’t have to go get said file from the hard drive every time you ask for it. This is used to speed up the process of finding files.

Client

Client is a term used to distinguish the two computers in an interaction between a server and another device. For example, your computer is the client that asks for data from a server.

Database

A database is a file that contains information that is organized in a certain way, which can be defined by whoever created the database. It is similar to a dictionary, where words all have a definition and are organized in alphabetical order.

Data Breach

A data breach is a term which is used to define when an intruder manages to get a hold of confidential data in a database, which is supposed to be secured and inaccessible to the public.

Domain

A Domain is the base name of a website address. For example, marketplace or messenger all share the Facebook domain.

FTP

FTP is a protocol used to transfer files from a server to a client. If the server asked for identification, the client needs to login before requesting a file. The secure version of this protocol is FTPS.

Hacker

A hacker is someone who manages to enter a network or device that isn't supposed to be accessible. For example, people aren't supposed to use your computer since it belongs to you, but a hacker will find a way to access its content without your knowledge.

HTML Attribute

An Attribute is an umbrella term for all of the things you add to an HTML element like: class, id, style, and so on.

HTML Class

A class is what is used to group multiple elements of an HTML page. Let's say you want multiple images to have the same outline, they will use the same class.

HTML ID

It works like a class, but you only put it on one element of an HTML page.

JavaScript

JavaScript is a programming language that is typically used for web applications. It can handle the interactions with the server and is responsible for all of the animated parts of a web page.

Internet

The Internet is an international network of computers used to communicate and exchange data.

IP Address

An IP (Internet Protocol) address is a sequence of numbers divided by separators identifying a computer. For instance, an IPv4 (IP version 4) address, such as 192.140.235.235, is composed of four blocks of three numbers separated by periods.

Modal-View-Controller (MVC)

MVC is a software architecture pattern used to develop web applications. The “M” stands for Modal. The modal is the structure of the data used. The “V” stands for view. A view is a visual interface representing the data. Often, views are web pages. The “C” stands for controller. A controller enumerates the actions a user can do, and it determines how to manipulate the data.

Network

A network is a group of computers linked together so that they can communicate and exchange data. There are two types of networks. Local area network (LAN) connects computers in the same building, whereas Wide area networks (WAN) can connect distant computers. The internet is a WAN.

Programming Language

A programming language is a system of notation used to create computer programs. It is used to define a process, dynamically use data and determine a computer’s behaviour. C, C++, C#, Java, JavaScript, PHP, Swift, Python and many other languages are programming languages.

Password

A password is a sequence of characters used to connect to networks and websites. A password should only be known by its creator.

Packets

A packet is a small segment of a message. After reaching their destination, packets are reassembled into the original message.

Phishing

Phishing is an attempt to trick people into clicking fake links in emails. Hackers using this tactic act as a bank or an online shop. Often, this link leads to a fake site stealing the target’s credentials.

Virus

A virus is a piece of software infecting a computer that can delete files, track data, or take over a computer. Viruses make their way into computers when a computer’s owner installs malicious files or clicks on suspicious links.

Web Browser

A web browser is a program used to navigate the web and read web pages. The most popular web browsers are Google Chrome, Safari, Microsoft Edge and Firefox.

Web Root

A web root is the topmost public directory containing web files in a website’s server.

Website

A website is a collection of related web pages under a certain domain. It is often hosted by a single organization.
